# study_runtime

A developer-themed productivity operating system for exam preparation. Plan, track, and compile knowledge with AI-powered study pipelines.

## Features

- **Dashboard** — System monitor for study metrics, weekly hours, and exam queue
- **Exam Manager** — Track exams with syllabus progress meters and priority flags
- **Schedule Generator** — AI engine that builds optimized study timelines based on difficulty
- **Focus Engine** — Pomodoro-based concentration timer with terminal aesthetic
- **Task Pipeline** — Organize tasks with priority sorting, due dates, and recurring jobs
- **Study Logger** — Log sessions with productivity metrics and mood tracking
- **Knowledge Base** — Notes system with tags, search indexing, and pinning
- **Analytics Engine** — Performance metrics with chart visualizations and readiness scores
- **Calendar** — Sprint-style calendar for exams, study blocks, and revision
- **Achievements** — Unlock badges for streaks, study hours, and task completions
- **XP System** — Level progression with experience points and streak tracking
- **Landing Page** — Developer onboarding with feature showcase and system metrics

## Tech Stack

- React 18 + TypeScript
- date-fns
- CSS (dark-first design system with progressive responsive breakpoints)
- LocalStorage (persistence)
- Vite
