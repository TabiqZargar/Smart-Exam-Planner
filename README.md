# Smart Exam Planner

Live Demo:
https://smart-exam-planner.vercel.app

## Overview

Smart Exam Planner is a web application that helps students create a study schedule before exams.

Users can:

- Enter exam date
- Add subjects
- Set daily study hours
- Generate a study plan
- Track progress
- Save data locally

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- LocalStorage
- Vercel

## Why I Built This

As a BCA student, I often needed a simple way to organize exam preparation and revision schedules.
